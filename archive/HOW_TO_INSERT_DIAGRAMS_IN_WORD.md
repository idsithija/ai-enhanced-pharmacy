## 🎨 Formatting Tips for Professional Look

### Centering the Diagram
1. Click on the inserted image
2. Go to **Layout Options** (icon next to image)
3. Select **In Line with Text**
4. Press `Ctrl+E` to center

### Adding a Caption
1. Right-click the diagram
2. Select **Insert Caption**
3. Choose **Figure** as label
4. Type caption (e.g., "Figure 5.1: System Architecture Diagram")
5. Click **OK**

### Wrapping Text
1. Click the diagram
2. **Layout Options** → Select **Square** or **Tight**
3. Choose **Centered** alignment

### Maintaining Quality
- ✅ **Do**: Keep original resolution (1920×1080)
- ✅ **Do**: Use corner handles to resize (maintains aspect ratio)
- ❌ **Don't**: Stretch or squash (use middle handles)
- ❌ **Don't**: Compress images (uncheck "Compress images" in Word options)

## 💡 Pro Tips

### 1. Consistent Sizing
- Use same width for all diagrams (e.g., 6.5 inches for landscape A4)
- Maintain aspect ratio by dragging corners only

### 2. Page Breaks
- Add page break before large diagrams: `Ctrl+Enter`
- Prevents diagrams from splitting across pages

### 3. Cross-References
```
"As shown in Figure 5.1, the system follows a three-tier architecture..."
```
To create cross-reference:
1. **Insert** → **Cross-reference**
2. Select **Figure**
3. Choose the figure number
4. Click **Insert**

### 4. Table of Figures (Automatic)
1. Place cursor where you want the list
2. **References** → **Insert Table of Figures**
3. Select **Figure** as caption label
4. Click **OK**
5. Auto-generates: "Figure 5.1: System Architecture... Page 23"

### 5. High-Quality Printing
Before printing/PDF export:
1. **File** → **Options** → **Advanced**
2. Scroll to **Image Size and Quality**
3. Uncheck **"Compress images in file"**
4. Set to **"High fidelity"**

## 📤 Exporting to PDF (High Quality)

### Method 1: Save as PDF
1. **File** → **Save As**
2. Choose **PDF** as file type
3. Click **Options**
4. Select **"ISO 19005-1 compliant (PDF/A)"** for archival
5. Click **OK** → **Save**

### Method 2: Export
1. **File** → **Export** → **Create PDF/XPS**
2. Choose **Standard (publishing online and printing)**
3. Click **Publish**

## 🔍 Troubleshooting

### Issue: Image looks blurry in Word
**Solution**: 
1. Right-click image → **Format Picture**
2. Go to **Picture** → **Picture Corrections**
3. Set **Sharpness** to +25%
4. Ensure compression is disabled (see Pro Tip 5)

### Issue: Image is too large
**Solution**:
1. Click image
2. Drag corner handle (not side/top handles)
3. Recommended width: 6.5 inches (fits A4 portrait)

### Issue: Caption numbering is wrong
**Solution**:
1. Right-click caption
2. **Update Field** (press `F9`)
3. Or: Select all (`Ctrl+A`) → Right-click → **Update Field**

### Issue: Image moves when typing
**Solution**:
1. Click image
2. **Layout Options** → **Fix position on page**
3. Or: Use **In Line with Text** for stable positioning
